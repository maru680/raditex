# Raditex Workspace

Claude Code workspace template for Raditex IT consultancy.

## Quick Start

1. Open this folder in VS Code
2. Install Claude Code extension
3. Claude will automatically read `CLAUDE.md` on startup

## Available Commands

| Command | Description |
|---|---|
| `/new-proposal` | Generate a new client proposal |
| `/site-review` | Review the website for issues |
| `/translate` | Translate content (ET/EN/DE) |

## Structure

- `CLAUDE.md` — main instructions for Claude Code
- `.claude/commands/` — custom slash commands
- `src/` — website source files
- `docs/` — proposals, assessments, templates
- `assets/` — images and media
