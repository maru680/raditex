# Generate New Client Proposal

Create a professional IT service proposal for a new client.

## Steps
1. Ask for the following if not provided:
   - Client company name
   - Contact person name and title
   - Services requested (see CLAUDE.md for service list)
   - Approximate company size (number of employees)
   - Any specific pain points or requirements

2. Generate a .docx proposal using Node.js (docx library)

3. Structure:
   - Cover page with Raditex logo placeholder and client name
   - Executive summary (2-3 sentences)
   - Proposed services with descriptions
   - Pricing table in EUR
   - Timeline
   - Next steps
   - Raditex OÜ contact footer

4. Save to /docs/proposals/YYYY-MM-DD_ClientName_proposal.docx

## Language
Always write proposals in Estonian unless client is foreign.
