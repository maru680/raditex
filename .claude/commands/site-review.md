# Website Review

Review the Raditex website for quality, performance, and content issues.

## Check List
1. **HTML structure** — semantic tags, heading hierarchy, missing alt texts
2. **CSS** — unused styles, mobile responsiveness, CSS variable usage
3. **Content** — typos, outdated info, unclear value propositions
4. **Performance** — large images, render-blocking scripts
5. **SEO basics** — meta tags, title tags, structured headings
6. **Accessibility** — contrast ratios, keyboard navigation, aria labels

## Output
Provide a prioritized list of issues:
- 🔴 Critical (broken, missing, or misleading)
- 🟡 Important (affects UX or SEO)
- 🟢 Nice to have (polish and optimization)
